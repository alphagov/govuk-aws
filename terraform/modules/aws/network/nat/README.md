## Module: aws/network/nat

Create a NAT gateway and associated EIP on each one of the public
subnets provided.

## Requirements

No requirements.

| Name | Version |
|------|---------|
| <a name="provider_aws"></a> [aws](#provider\_aws) | n/a |
